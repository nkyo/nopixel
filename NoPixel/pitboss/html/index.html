Dealer Instructions

  • /bj Open table - players can now place bets
  • /bj deal Start game and deal cards - player's bets are locked and cards are dealt
  • /bj hit [Player Number] Deal player a card. "/bj hit d" or "/bj hit dealer" for dealer.
  • /bj double [Player Number] Doubles player's bet, but only gets one hit. If not enough chips left, will "double" for remaining chips.
  • /bj end This will end the current game, and clear all card HUDs. Will not calculate winning/losing if ended early.
  • /bj chips [0 | 1] 0: Manually handle cash each hand
    1: Only allow bets for players who have purchased chips. Auto pays/takes chips. See "Player Commands" below. (default)
  • /bj decksize [number of decks] Change # of decks. Use higher # for more players. (Default 4).
  • /bj shuffle Shuffle current deck. Game will shuffle when 33% deck remains.
  • /bj maxbet [number] Set max allowed bet. Defaults to 1000 when table is opened.
  • Player Commands
  • /pokerchips buy [dealer ID] [amount] to buy chips
  • /pokerchips cash [dealer ID] [amount] to redeem chips
  • /pokerchips accept (Dealer) to accept a request. This will send/receive cash for chips
  • /pokerchips will reveal balances. Dealer will see negative values for players that currently have remaining chips.
  • /bj leave for players to clear their cards and get out of any 'invisible' games if they exist
